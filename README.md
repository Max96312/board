# 게시판 서비스
처음 부터 끝까지 하나의 서비스를 기획하고 설계하며 구현하는것에 목적을 두어 개발하였습니다.
1인 개발이며 사이드 프로젝트로 진행하는 프로젝트라 Thymeleaf와 bootstrap을 사용하였습니다.

1명
